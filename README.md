# GithubSearch
This project the  users can enter a GitHub username into a form, submit it, and see names and descriptions of that person's public repositories. A person can also look for repositories.

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 12.1.2.


## Built With

- HTML
- CSS
- Bootstrap
- Javascript
- Typescript
- Angular CLI version 12.1.2

## Getting Started

To get a local copy, follow these simple example steps.

### Prerequisites

A computer with a working and up to date web browser.

### Setup

- Clone the repository to your folder of choice using Git (or just download all the files)

```
$ git clone https://github.com/Barrack-coder/Github-search.git

```

- Open the index.html file (in the root folder) and the website should show up.
